# Employee Salary Management System

This Java-based application allows users to calculate and manage employee salaries using Object-Oriented Programming concepts. 
It supports different employee types (Manager and Staff) with inheritance and provides interactive console input for entering employee details.

## Features
- Add multiple employees with custom details
- Manager salaries include bonuses
- Staff salaries include extra hours calculation
- Interactive menu for displaying employee information
- Robust input validation

## Technologies
- Java (OOP)
- Eclipse IDE
